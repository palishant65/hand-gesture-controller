[7:05 pm, 29/12/2025] Bhaiya: # Hand Gesture Controller

A real-time hand gesture recognition project using Python, OpenCV, and MediaPipe.
The system detects hand gestures through a webcam and can be used for gesture-based control systems.

## Technologies
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
pip install -r requirements.txt
python collect_data.py
python hand_3D_animation.py

## Features
- Real-time hand detection
- Multiple gesture recognition
- Webcam-based interaction
